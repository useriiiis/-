📚 Elliott Wave Theory Quick Guide
Impulse Waves (1→2→3→4→5): The main trend movement. Wave 3 is usually the longest. Wave 4 cannot enter Wave 1 territory.
Corrective Waves (A→B→C): Counter-trend movement after impulse completes. Usually 50-78.6% retracement of waves 1-5.
Fibonacci Targets: Wave 3 = 1.618× Wave 1; Wave 5 = 0.618× to 1× of Wave 1+3; Wave C = 1× Wave A.
